# Mnemonic

A Godot addon that automatically captures and organizes video, audio, and editor context throughout development for devlogs.

## Install

1. Download the latest release from [GitHub Releases](https://github.com/bioartgames/mnemonic/releases).
2. Extract the zip and copy `mnemonic/` into your Godot project's `addons/` folder.
3. Open your project in Godot → **Project → Plugins** → enable **Mnemonic**.
4. Click **Start recording** in the Mnemonic dock.

See [`addons/mnemonic/INSTALL.md`](addons/mnemonic/INSTALL.md) for details.

## Documentation

https://bioartgames.github.io/mnemonic/

## Requirements

- Windows 10 or later
- Godot 4.6+ (editor)

## Bundled Core

Release zips include a self-contained `Mnemonic.Windows.exe` and FFmpeg under `addons/mnemonic/core/`. Core is not built from this repository.
